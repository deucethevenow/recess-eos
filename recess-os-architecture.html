Request Type Routing

Each request type triggers a different Asana Rule that creates a discovery subtask with specific instructions.

Request TypeAuto-Created SubtaskAssigned ToSkill
Bug Complete Quick Discovery (Bug) Engineering lead /quick-discovery
Feature Complete Discovery (Feature) Product / Engineering /quick-discovery → /discovery if L/XL
New Project Complete Deep Discovery Product lead /discovery (full)
Issue Complete Discovery (Issue) Dept lead / COO /quick-discovery with 5 Whys
Opportunity Complete Discovery (Opportunity) Strategy / COO /quick-discovery with value estimate
